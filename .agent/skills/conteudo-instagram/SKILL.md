## FORMATOS DISPONÍVEIS

### 1. CARROSSEL (O formato que o Evandro mais gosta)
**Estilo preferido:** Carrosseis "Sniper" — diretos, com design limpo e copy forte.

**Estrutura padrão:**
```
Slide 1: HOOK (gancho que para o scroll)
    → Pergunta provocativa OU declaração contraintuitiva
    → Design impactante, texto grande e legível
    
Slide 2-6: CONTEÚDO (valor + educação)
    → 1 insight por slide
    → Textos curtos (máx 3 frases por slide)
    → Ícones ou elementos visuais de apoio
    
Slide 7: CTA (chamada para ação)
    → "Salve para consultar depois"
    → "Compartilhe com alguém que precisa"
    → "Link na bio para [algo específico]"
```

### 2. REELS (Vídeos Curtos)
**Roteiro de Reel Viral:**
```
[0-1s] HOOK VISUAL (movimento, algo inesperado)
[1-3s] HOOK VERBAL ("Você sabia que...")
[3-15s] CONTEÚDO (entrega rápida de valor)
[15-20s] TWIST/SURPRESA (algo inesperado para manter)
[20-30s] CTA ("Comente X para receber...")
```

### 3. STORIES DE VENDAS
**Roteiro Fênix de Stories (sequência de 8-12 stories):**
```
Story 1: CONTEXTO (o que está acontecendo hoje)
Story 2: DOR (identificação com o problema)
Story 3-4: AGITAÇÃO (por que as soluções comuns não funcionam)
Story 5-6: SOLUÇÃO (introduzir o produto/serviço)
Story 7-8: PROVA (depoimento, resultado, screenshot)
Story 9: OFERTA (o que está disponível)
Story 10: CTA (enquete/link/DM "Quero")
```
